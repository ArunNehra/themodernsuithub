# The Modern Suit Hub (Hansi, Haryana)

A premium, mobile-first web application for Hansi's first AI-powered ladies suit boutique. 
This project leverages a zero-cost infrastructure architecture designed to handle thousands of daily visitors with flat ₹0/month operational costs.

### Key Features:
* 🛍️ **Interactive Catalog & Filters:** Filter suits by Occasion, Budget, Color, Fabric, and Work with zero lag.
* 👗 **AI Stylist Quiz:** A visual step-by-step styling advisor that matches user attributes to catalog items.
* 🪞 **Virtual Try-On Canvas:** Selfie upload tool running CatVTON image generation with a smooth WhatsApp fallback.
* 🪙 **Coins & Referrals:** WhatsApp-based frictionless login, rewards ledger, and coupon link generators.
* 🔐 **Admin Control Panel:** Passcode-secured suite (`Hansi2026`) for managing catalog items and confirming orders directly into Google Sheets.
